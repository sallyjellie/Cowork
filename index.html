📋 Product Summary — Key Updates
🧠 Model Strategy — Multi-Model by Design ▼

Copilot Cowork is built on a multi-model architecture, enabling flexibility and cost optimisation across different task types.

Generally Available

  • Anthropic Opus 4.8 — for complex, reasoning-heavy tasks
  • Anthropic Sonnet 4.6 — balanced performance and speed
  • Sonnet + Opus Advisor — adaptive model routing

Frontier (Early Access)

  • GPT 5.5 (Frontier) — hosted on Azure; data stays within Microsoft. Covered by Azure enterprise security, compliance certifications, and data-residency controls. Available in EU (EUDB) and China/Hong Kong.

Coming Soon

  • Cowork 1 — a secure, fine-tuned model designed to handle everyday Copilot tasks at a substantially lower cost. Removes model bias and is optimised for enterprise-grade workloads, making it ideal for cost-sensitive, high-volume use cases.

💳 Pricing Model — 3 Ways to Access Copilot Credits ▼

Copilot Cowork requires the Microsoft 365 Copilot User Subscription Licence (USL). Cowork usage is then billed via Copilot Credits — the universal currency for M365 AI services. There are three ways to purchase credits:

Pay-As-You-Go (PAYG) Capacity Packs Pre-Purchase Plan (P3)
Ideal for No prior usage history; getting started or testing Steady, predictable usage Larger or variable demand
Price $0.01 / Copilot Credit $200 per pack / month / tenant 3 tiers, bulk volume discount
Purchase construct Pay-as-you-go, monthly Licence subscription (upfront — 3 yr) One-time Azure purchase (annual upfront)
Where to purchase M365 Admin Center M365 Admin Center, Power Platform Admin Center Azure Portal
MACC eligible ✅ Yes ✅ Yes* ✅ Yes*

⚠️ Capacity Pack note: Unused monthly credits reset at the end of each month — plan usage accordingly.

Credits are charged based on 4 inputs: model use, context retrieval, tool calls, and runtime. Existing Capacity Packs can also be applied to Cowork.

🛡️ Control & Visibility for Admins ▼

🔒 Control

  • Off by default. Admins decide when to enable Cowork and who gets access.
  • Spending limits at tenant, group, and user levels. Hard caps and scoped access.
  • Custom usage alerts. Set thresholds by org or group; choose who gets notified.
  • User credit requests. Users can request additional credits from within Cowork when needed.

📊 Visibility

  • Usage reporting broken down by user, group, and feature — full accountability across the org.
  • Per-task pricing in credits (coming soon after GA) — users see the cost of each task as they run it.
  • Cost Management Dashboard in the M365 Admin Center — single pane for all usage-based billing.
